## Работать со списками
Чтобы добавить не нумерованные списки,необходимо пункты выделить звездочкой(*).
Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованые  писки, необходимо просто пронумероват.
Например вот так:
1. Список
2. Список
3. Список
